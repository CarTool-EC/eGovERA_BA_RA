Fraunhofer Omejdn Server ()
Description: An OAuth2/OpenID Connect server for IoT devices which use their private keys to request OAuth2 access tokens in order to access protected resources and websites or apps which retrieve user attributes. Additional information: Omejdn is a lightweight server that can be easily deployed on a Raspberry Pi or other small device. It is designed to be easy to use and configure, and it provides a secure way for IoT devices to access protected resources. Example: Omejdn can be used to enable IoT devices to access cloud-based services, such as data storage or data analytics. It can also be used to enable IoT devices to communicate with each other securely.
