Translation Item Metadata ()
Definition: Translation Item Metadata refers to structured information that provides descriptive and contextual details about a translation item or unit of content in a translation process. It includes attributes such as the item's source language, target language, content type, translation status, translator information, and any additional metadata that helps define its characteristics and supports the translation workflow. Translation Item Metadata enables efficient management, tracking, and quality control of translation tasks, ensuring accuracy and consistency across multilingual content.
