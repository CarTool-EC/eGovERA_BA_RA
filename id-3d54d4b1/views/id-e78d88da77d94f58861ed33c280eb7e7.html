TVI-Identity and Access Infrastructure Enablers [Motivation] ()
NARRATIVE: The TVI-Identity and Access Infrastructure Enablers [Motivation] view extends Technical view - infrastructure and provides a comprehensive framework for leveraging various components and services that facilitate identity and access management solutions, security controls, and privacy-related functionality within an application or system. The view is divided into four main groupings: Identity and Access Infrastructure Enablers, Security Enablers, Management Infrastructure Enablers, and Privacy Enablers. The Identity and Access Infrastructure Enablers grouping includes Technology Service ABBs like Identity Provider, Federated Identity Provider, Active Directory, and Lightweight Directory Access Protocol. These services provide the underlying functionality to implement identity and access management solutions, including user authentication, access to online resources, and maintaining distributed directory information services over an IP network. The Security Enablers grouping includes Application Service ABBs and Application Component ABBs such as Audit, Authentication, Registration, Identification, Access Management, Identity Management, Authorisation, Accounting, and Request Validation. These components enable the implementation and management of security controls and measures, ensuring the correct behaviour, security, and compliance of different building blocks. The Management Infrastructure Enablers grouping includes Technology Service ABBs like Logger, Audit Manager, Telemetry, and Management Console. These services enable efficient management of public administration IT platforms, including recording system events, managing audits, tracking performance, and identifying issues. The Privacy Enablers grouping includes Application Service ABB and Application Component ABB named Privacy. These components facilitate the implementation of privacy-related functionality within an application or system, such as storing, securing, anonymising, pseudonymising, rectifying, and erasing personal data. The motivation behind this view is to provide a comprehensive framework that leverages various components and services to ensure secure and efficient identity and access management, security controls, and privacy-related functionality within an application or system.
