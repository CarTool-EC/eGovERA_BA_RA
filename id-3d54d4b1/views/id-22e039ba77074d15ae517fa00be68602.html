SV-Business Agnostic High Value Dataset ()
NARRATIVE: The SV-Business Agnostic Dataset view extends the technical view application and provides a comprehensive framework for leveraging various high-value datasets in a business-agnostic manner. The view consists of two main elements: the Business Agnostic Dataset and the Business Agnostic Business Information Base. 1. Business Agnostic Dataset: - [Geospatial High Value Dataset]: This dataset comprises INSPIRE themes, reference parcels, and agricultural parcels. It is a data object that represents geospatial information. - [Business-related data High Value Dataset]: This dataset comprises basic company information, records, and financial statements. It is a data object that represents business-related data. - [Meteorology High Value Dataset]: This dataset comprises observation measurement data from weather stations, climate data, weather warnings, radar data, and NWP model data. It is a data object that represents meteorological information. - [Mobility High Value Dataset]: This dataset comprises INSPIRE themes and River Information Services data. It is a data object that represents mobility-related information. - [Statistics High Value Dataset]: This dataset comprises statistical data on select legal acts. It is a data object that represents statistical information. - [Earth Observation Data High Value Dataset]: This dataset comprises INSPIRE themes, environmental information according to the environmental information directive, and datasets related to select legal acts. It is a data object that represents earth observation data. 2. Business Agnostic Business Information Base: - [Availability]: This business object represents the availability of the use of a predefined object or space and the future demand. - [Data Quality]: This business object represents the state of qualitative or quantitative pieces of information. - [Document]: This business object represents a written, drawn, presented, or memorialized representation of thought, often the manifestation of non-fictional, as well as fictional, content. The elements within the Business Agnostic Dataset are grouped under the Grouping ABB type, while the elements within the Business Agnostic Business Information Base are grouped under the Diagram Model-group ABB type. The motivation behind this view is to provide a structured representation of high-value datasets and business information that can be used in a business-agnostic manner. By separating the datasets from the business information, organizations can leverage these datasets for various purposes without being tied to specific business contexts. This allows for greater flexibility and reusability of the datasets across different domains and applications. Additionally, the inclusion of business objects such as availability, data quality, and documents provides a holistic view of the business information base that supports the utilization of the datasets.
