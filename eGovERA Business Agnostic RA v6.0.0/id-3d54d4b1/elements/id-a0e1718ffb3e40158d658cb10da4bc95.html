Definition: Knowledge discovery Enablers ABB is a Grouping that refers to components and services that facilitate the process of discovering insights and knowledge from data Source: European Comission-Open Science Source reference: https://research-and-innovation.ec.europa.eu/strategy/strategy-2020-2024/our-digital-future/open-science_en Example: The following implementation is an example on how this specific Architecture Building Block (ABB) can be instantiated as a Solution Building Block (SBB): CELLAR. Cellar metadata is stored in a knowledge graph, and more specifically in a triple store.
