dct:title2022 Strengthened Code of Practice on Disinformation
dct:descriptionThe Strengthened Code of Practice on Disinformation is an updated version of the code that aims to combat the spread of false information online. Introduced in 2022, it includes stricter guidelines for tech companies and social media platforms to tackle disinformation effectively. The code emphasizes the need for transparency, accountability, and cooperation between platforms, governments, and fact-checking organizations. It encourages platforms to take proactive measures to identify and remove disinformation, promote reliable sources, and limit the reach of misleading content. The strengthened code also promotes user empowerment by providing tools to report and flag false information. Overall, it seeks to create a safer and more trustworthy online environment by addressing the challenges posed by disinformation.
