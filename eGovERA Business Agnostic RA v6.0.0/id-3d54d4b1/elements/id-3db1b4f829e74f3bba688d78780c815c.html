RDF-XML ()
Description: RDF-XML (Resource Description Framework XML) is a syntax format used to represent metadata and structured data in the form of Resource Description Framework (RDF) triples. RDF-XML provides a way to encode and exchange data using XML (eXtensible Markup Language) syntax.
