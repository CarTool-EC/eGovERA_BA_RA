Transaction metadata ()
Definition: Transaction Metadata refers to structured information that captures details and contextual information about a specific transaction. It includes attributes such as transaction ID, date and time, parties involved, transaction type, transaction status, and any additional metadata relevant to the transaction. Transaction Metadata provides a comprehensive view of the transaction, enabling data analysis, auditing, tracking, and reporting.
