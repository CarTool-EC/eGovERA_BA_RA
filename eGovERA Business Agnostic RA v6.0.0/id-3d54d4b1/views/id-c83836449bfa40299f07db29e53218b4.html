TVI-Computing Infrastructure Enablers [Motivation] ()
NARRATIVE: The TVI-Computing Infrastructure Enablers [Motivation] view extends Technical view - infrastructure and provides a comprehensive framework for leveraging various components and services that facilitate the development, deployment, and management of virtual and computing infrastructure. This view is divided into several groups, each containing different elements that contribute to the overall computing infrastructure. The first group, "Computing Infrastructure Enablers", includes two Node ABBs: "Virtual Machine" and "Server". The Virtual Machine ABB enables the emulation of a physical computer to run programs and deploy applications, while the Server ABB enables access to resources or services to other computers or devices on a network. The second group, "Artificial Intelligence Enablers", contains several Application Service ABBs and Application Component ABBs that provide the necessary infrastructure and resources for AI developers to build intelligent systems. These include "Artificial Intelligence", "Machine learning", "Artificial Intelligence Engine", and "Natural-Language Processing (NLP)". The third group, "Analytics Enablers", refers to components and services that serve any current or future Digital Service Infrastructure (DSI) requiring data analysis capabilities. This group includes "Data Analytics", "Big Data Analytics", "Business Intelligence", and "Smart queue management". The fourth group, "Data Management Enablers", refers to components and services that support effective data management practices and procedures. This group includes "Data Publication", "Forms Management", "Content Management", "Data Management", "e-Archiving", "Metadata Management", "Data Quality", "Data Extraction, Transformation, and Loading", "Data Persistence", "Data Virtualization", "Multimedia distribution", "Submission Service", "Transaction Management", "Archival Information Packages", "Trusted Digital Repositories", "Service Metadata Publisher", "CRM", "Data Publication", "Forms Management", "Data Extraction, Transformation and Loading", "Data Virtualization", "eFiles Management", "Service Metadata Locator", "Open Data Management", "Contact Management". The fifth group, "Orchestration Enablers", refers to the components and practices that support the development and deployment of orchestration applications. This group includes "Orchestration" and "Workflow Management". The final group, "Data Exchange Enablers", refers to tools or components that facilitate the exchange of data between systems or applications. This group includes "Data Exchange", "Once-Only Technical System", and "Data Exchange". Each of these groups and their elements play a crucial role in the overall computing infrastructure, providing the necessary tools and services to facilitate the development, deployment, and management of various applications and systems.
